# Interesta Homebrew Tap

Official package metadata and release binaries for the Interesta Orchestrator
CLI. This repository does not contain the private CLI source code.

## Install

On macOS or Linux:

```sh
brew install interesta-ai/tap/orchestrator
```

On Windows:

```powershell
winget install --id Interesta.Orchestrator --exact
```
